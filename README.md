# 楽天EC 売上・広告分析ダッシュボード

## 概要
楽天ECを想定した売上・広告分析ダッシュボードです。  
Power BIを使用して、売上、広告費、ROASを可視化しました。

## 使用ツール
- Power BI
- Excel
- GitHub

## 実装内容
- 注文データ、広告データ、商品マスタの取り込み
- Calendarテーブル作成
- DAXメジャー作成
  - Total Sales
  - Total Ad Cost
  - ROAS
- SKU別売上・広告費・ROAS分析
- 月別売上・広告費・ROAS比較

## 主な分析結果
- 2025年2月は1月と比較して広告費を抑えながら売上が増加し、ROASが改善した
- SKU別ではSKU002〜SKU003が高い広告効率を示した
- 低ROASの商品を把握し、改善対象の洗い出しが可能

## 工夫した点
- 日付テーブルを作成し、月別分析ができるようにした
- 商品マスタ、注文データ、広告データをリレーションで接続した
- KPIカード、SKU別表、月別表、グラフを組み合わせて見やすくした
